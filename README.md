# My Profile

A mobile application that represents my professional journey. It's built with React Native and Expo.



Une application mobile qui représente mon parcours professionnel. Elle est faite en React Native avec Expo.


## Install the Expo Go app on your phone / Installer l'application Expo Go sur le téléphone

* Android : Download from the Google Play Store

* IOS : Download from the Apple Store



* Android : Téléchargez depuis le Google Play Store

* IOS : Téléchargez depuis l'Apple Store


## Launch the project / Lancer le projet

Open the terminal then the project `cd profile`. Type the command `npx expo start --tunnel`.

* Android : Open the Scan QR on Expo app and scan the QR code that appears on the terminal.

* IOS : Open camera and scan the QR code that appears on the terminal.



* Android : Ouvrez l'application Scanner de QR sur l'application Expo et scannez le QR code  qui s'affiche sur le terminal.

* IOS : Ouvrez l'appareil photo et scannez le QR code qui apparaît sur le terminal.
